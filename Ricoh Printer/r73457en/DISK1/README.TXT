 2) Getting started
 
  2-1) Installing the printer driver 
   2-1-1 Windows 7/Vista/Server 2008 R2
    Be sure to close all applications before installing the driver.	
    Note: This procedure requires administrator privileges (admin account or "Run as Administrator").
      2-1-1-a Standard installation
          Step 1 From the "Start" menu, select "Devices and Printers" > "Add a Printer".
          Step 2 The "Add a Printer" dialogue will be displayed asking "What type of printer do you want to 
                       install?" Select "Add network, wireless, or Bluetooth printer".
          Step 3 The "Searching for available printers" dialogue appears. Select the gDEVICEh from the list 

                       of printers and click "Next". (If you do not see your gDEVICEh in this list, please proceed to 

                       "What to do if your device is not found" at the end of this section.)
          Step 4 The "Install the printer driver" dialogue appears. Click "Have Disk".
          Step 5 Browse to the folder containing the driver files (typically the CD-ROM or an extracted archive 

                       downloaded from the website) and click "OK".
          Step 6 Follow the rest of the Wizard's instructions to complete the installation.
      2-1-1-b What to do if your device is not found
          Step 1 From the "Start" menu, select "Devices and Printers" > "Add a Printer".
          Step 2 The "Add a Printer" dialogue will be displayed asking "What type of printer do you want to 

                       install?" Select "Add a local printer".
          Step 3 The "Choose a printer port" dialogue will be displayed. Create a new "Standard TCP/IP Port" 

                       and click "Next".
          Step 4 Enter the "DEVICE" IP address and uncheck "Query the printer and automatically select the 

                       driver to use". Click "Next".
          Step 5 The "Install the printer driver" dialogue appears. Click "Have Disk".
          Step 6 Browse to the folder containing the driver files (typically the CD-ROM or an extracted archive 

                       downloaded from the website) and click "OK".
          Step 7 Follow the rest of the Wizard's instructions to complete the installation.

   2-1-2 Windows XP and 2K
    Be sure to close all applications before installing the driver.
    Note: This procedure requires administrator privileges
          Step 1 Open gPrinters and Faxes from the Start menu.
          Step 2 Select gAdd Printerh.
          Step 3 gWelcome to the Add Printer Wizardh is displayed. Click gNext >h.
          Step 4 The gLocal or Network Printerh dialogue is displayed. Select gLocal printer attached to this 
                       computerh and click gNext >h. Note: Do not enable gAutomatically detect and install my Plug 
                       and Play printerh.
          Step 5 The gSelect a Printer Porth dialogue is displayed. Select gCreate a new porth > gStandard 
                       TCP/IP Porth as the port type and click gNext >h.
          Step 6 The gAdd Standard TCP/IP Printer Port Wizardh is displayed. Click gNext >h.
          Step 7 Enter the device IP address or hostname and click gNext >h.
          Step 8 The gInstall Printer Softwareh dialogue is displayed. Select gHave Disk ch.
          Step 9 Browse to the folder containing the driver files (typically the CD-ROM or an extracted archive 
                       downloaded from the website) and click "OK".
          Step 10 Follow the rest of the Wizard's instructions to complete the installation.


  2-2) Upgrading the driver
   2-2-1 Windows 7/Vista/Server 2008 R2
    Be sure to close all applications before upgrading the driver.	
    Note: This procedure requires administrator privileges (admin account or "Run as Administrator").
          Step 1) Open the "Printers" folder (or "Printers and Faxes") from the "Start" menu or "Control Panel". 
          Step 2) Open the Properties of the printer to be upgraded.
          Step 3) Advanced tab > New Driver button > Next button.
          Step 4) Select "Have Disk..." then browse to the folder containing the extracted printer driver INF 
                                 file. Click Open. 
          Step 5) Click OK and select the printer model. Click Next.
          Step 6) Once the installation is complete, click Finish.
          Step 7) Click OK to close the Printer Properties dialog box.


  2-3) Uninstalling the printer driver 
   2-3-1 Windows 7/Vista/Server 2008 R2
    Be sure to close all applications before uninstalling the driver.	
    Note: This procedure requires administrator privileges (admin account or "Run as Administrator").
          Step 1) Control Panel > Printers 
          Step 2) Locate the logical printer corresponding to the driver that is to be uninstalled and delete it.
          Step 3) Delete the driver from Server Properties > Driver tab. 
          Step 4) Log the current user account off.


   2-3-2 Windows XP and 2K
    Be sure to close all applications before uninstalling the driver.
    Note: This procedure requires administrator privileges
          Step 1) Open the "Printers" folder (or "Printers and Faxes") from the "Start" menu or "Control Panel". 
          Step 2) Delete the logical printer corresponding to the driver that is to be uninstalled.
          Step 3) File > Server Properties > Driver tab.  
          Step 4) Select the driver to be uninstalled.  
          Step 5) Press the Delete key.  
          Step 6) Log the current user account off.


  3) Other important points

    -1 During the printer driver installation, Windows may alert that Windows
       can't verify the publisher of this driver.  However, the printer driver
       functionality is not affected.  Please continue the installation.

    *  Limitation/Notes in using PS printer driver

       [Windows functions]
    * If virtual memory on a Windows operating system is set lower than usual,
      insufficient memory errors will occur and drawing operations in
      applications or printer drivers will not perform properly.
  
      Documents might not print correctly or the following problems might occur:
        - Low memory warnings.
        - Application errors.
        - Printing discontinues without warning.
      Increase the memory setting for virtual memory, and then start printing.
      if this is not helpful, add additional memory to the computer.

    * If a Separator Page is selected, some documents may not print as expected.

    * Please do not change the "Driver" settings on "Advanced" tab.
    
    * When you set up an IPP Printer, please set
      "http://<Printer's IP Address>/ipp/port1" as Printer URL.
